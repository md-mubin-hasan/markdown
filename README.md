# Markdown Commands
This is a list of Markdown commands which I know
```
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
```
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
There are at most 5 heading styles possible
Bulleted list:
```
* Item 1
* Item 2
* Item 3
```
* Item 1
* Item 2
* Item 3
Numbered list:
```
1. Item 1
2. Item 2
3. Item 3
```
1. Item 1
2. Item 2
3. Item 3
